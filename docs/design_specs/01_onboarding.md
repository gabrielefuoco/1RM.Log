# Page Specification: App Entry & Onboarding

**Mockup Source**: [app_entry_&_onboarding_landing](../../mockup/app_entry_&_onboarding_landing/code.html)

## Overview
The first screen the user sees. Minimalist, high-impact branding.
> [!NOTE]
> Strict adherence to `#13ec6d` Neon Green and `#102218` Dark Background.

## UI Components

### 1. Background
- **Color**: `#102218` (Global Background Dark).
- **Ambient Effects**: Large blurs using `bg-primary/20` (Neon Green glow).

### 2. Branding
- **Logo**: Bars and Text in **Neon Green** (`text-primary`).
- **Headline**: White main text, **Neon Green** accent ("Risultati Scientifici.").

### 3. Actions
- **Primary Button**: `bg-primary text-background-dark` (Neon Green background, Dark text).
- **Secondary Button**: `border-primary/30 text-white`.

## Consistent Tokens
- `primary`: `#13ec6d`
- `background`: `#102218`
- `font`: `Lexend`
