# Page Specification: History & Configuration

**Mockup Source**: [training_calendar](../../mockup/training_calendar_&_history_1/code.html)

## Overview
Configuration for progression logic and history viewing.
> [!NOTE]
> All UI elements must use the Neon Green (`#13ec6d`) accent color.

## UI Components

### 1. Header
- **Icons**: `text-primary` (Magic Wand).

### 2. Progression Rules (Toggles & Sliders)
- **Switch/Toggle**:
  - Active: `bg-primary`.
  - Track: `bg-white/20`.
- **Range Slider**:
  - Thumb: White with `border-primary`.
  - Fill/Track: `bg-primary` (active portion) or `bg-white/10`.
- **Input Fields**:
  - Focus Ring: `ring-primary`.
  - Text: White.

### 3. Segmented Controls (Weight Rounding)
- **Selected**: `bg-primary text-background-dark font-bold`.
- **Unselected**: `bg-surface-dark text-slate-400`.

### 4. AI Insight Box
- **Container**: `bg-primary/10 border border-primary/20 rounded-2xl`.
- **Icon**: `text-primary`.
- **Text**: `text-primary/80` or White.

### 5. Exercise Modal
- **Add Button**: `bg-primary text-background-dark`.
- **Copy Button**: `bg-white/10 text-white`.
- **Highlights**: All tags and icons use `text-primary`.
